# CifrumAICource
Сервис для работы модели предиктивной аналитики

```git clone https://github.com/EvgeniiBabaev/CifrumAICource```

```cd CifrumAICource```

настройка:

```pip install --upgrade pip```

```pip install -r requirements.txt```

запуск:

```python main.py```

запуск в docker:

```docker-compose up -d --build```
